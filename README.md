# dotfiles

The repository is a long journey.

## Usage

### install

```
git clone git@github.com:asmz/dotfiles.git dotfiles
cd dotfiles
make
chsh -s /bin/zsh
```

### update

```
cd dotfiles
make update
```

## Requirements
* zsh
* make
* git
* curl
* vim
